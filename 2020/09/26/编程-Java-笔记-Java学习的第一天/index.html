0x00 下载JDK、安装JDK和配置环境变量

1.下载并安装JDK

https://www.oracle.com/java/technologies/javase-downloads.html

下载完成安装后，打开自己所安装的文件夹

2.配置环境变量

一般默认为C:\Program Files\Java\下会有一个（JDK-版本号）的文件夹

打开它目录下的bin文件夹，复制路径。

就像这样C:\Program Files\Java\jdk-11.0.8\bin

按下Win+E，右键此电脑→属性，在打开的新窗口中，选择左侧带盾牌图标的高级系统设置。

在高级选项卡中，点击右下角的环境变量按钮，下方有一个系统变量分组框。

下方列表中的变量列中有一项为Path，我们单击这一行，选择右下角的编辑按钮。

会弹出一个新的窗口，我们单击右侧的新建按钮，按下Ctrl+V将JDK的路径粘贴进去。

依次单击确定按钮关闭所有窗口。

3.验证

    1.按下Win+X，再按A键，打开PowerShell。

    2.输入javac，看是否输出帮助信息，输出帮助信息则为配置环境变量成功。
