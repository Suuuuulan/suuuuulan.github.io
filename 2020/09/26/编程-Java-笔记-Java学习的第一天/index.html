教程如上

故本帖会从0到1。（也包括安装，环境变量配置，或者自己平时遇到的点点滴滴的问题。）

    第一次初识Java应该是在以前的手机游戏上边（也包括一些APP），打开时界面上的一杯咖啡，还有图片下方带的“Java”字样，给我枯燥的童年留下了比较深刻的印象。

0x01 下载JDK、安装JDK和配置环境变量

1.下载并安装JDK

因为现在下载需要登录账号，建议先到www.oracle.com注册一个账号，再进行下载。

https://www.oracle.com/java/technologies/javase-downloads.html

下载完成安装后，打开自己所安装的文件夹，并记住所在位置。

2.配置环境变量

一般默认为C:\Program Files\Java\下会有一个（JDK-版本号）的文件夹

打开它目录下的bin文件夹，复制路径。

就像这样C:\Program Files\Java\jdk-14\bin

按下Win+E，右键此电脑→属性，在打开的新窗口中，选择左侧带盾牌图标的高级系统设置。

在高级选项卡中，点击右下角的环境变量按钮，下方有一个系统变量分组框。

下方列表中的变量列中有一项为Path，我们单击这一行，选择右下角的编辑按钮。

会弹出一个新的窗口，我们单击右侧的新建按钮，按下Ctrl+V将JDK的路径粘贴进去。

依次单击确定按钮关闭所有窗口。

3.验证

0x02 Eclipse的下载及汉化

在开始学习之前，我们应该为自己准备一个得心应手的工具，这里推荐eclipse，最主要是免费，其次也有汉化，对新手来说是个不错的选择！

1.下载

  1. 首先我们需要下载安装包，打开https://www.eclipse.org/downloads/packages/
  2. 找到Eclipse IDE for Java Developers，在右侧选择对应系统的版本。
  3. 选择过后将跳往新的网页，我们点击Download按钮下方的eclipse-java-**--R-win32-x86_64.zip
  4. 下载完成之后，将其中的eclipse文件夹解压出来，记好解压的位置。

2.汉化

  1. 打开https://www.eclipse.org/babel/downloads.php
  2. 找到下方的Babel Language Pack Zips
  3. 在这行话的下方一般会有3个日期，选择最新的。
  4. 在跳出的页面中，我们首先在选择语言，点击Chinese (Simplified)。
  5. 一般选择BabelLanguagePack-eclipse-zh_4.16.0.v20200711020001.zip (84.09%)后边为版本号不一定相同前缀选择正确即可。
  6. 依旧选择Download按钮下方的很长的链接。
  7. 下载完成之后，打开压缩包，再打开里边的eclipse文件夹，我们会发现还有两个文件夹。
  8. 将这两个文件夹解压到上述我们安装好的eclipse文件夹中。

3.新建快捷方式

​ 在eclipse文件夹中，有一个应用程序eclipse.exe，我们右键将快捷方式发送到桌面即可。

​ 双击打开就会发现一个汉化过的界面。

0x03 初识Eclipse IDE

​ 第一次打开，软件会提示你“选择一个目录作为工作空间”，这个自行选择存放的目录，左下角（将此值用作缺省值并且不在询问）我们选择打钩，下次打开eclipse就不会提示这个页面了。

​ 启动之后，我们会有一个欢迎页面，直接点击该标签右侧的关闭即可。如果打钩，每次开启都会有个欢迎页面。

关闭之后的界面

0x04 第一个程序

创建Java项目

  • 选择文件/新建/项目
  • 选择Java/Java项目/下一步
  • 输入项目名称/选择项目布局
  • 这里不用管，直接选择Don’t Create
  • 我们下一步是新建类，把HelloJava左侧的箭头点开，右键src/新建/类

创建成功

编写代码

既然项目名称已经是HelloWorld了，那就开始写吧！

运行

代码已经编写好了，让我们冲冲冲！！！！

我们可以直接按下Ctrl+F11，直接运行。

也可选择 菜单栏运行/运行

然后我们会发现我们在println()函数中的字符串是对应的，输出的结果也如我们所料。

0x05 更改编辑区的字体

在敲代码的过程中，长时间盯着屏幕，看小字是不是有模糊感，那么字体能不能调呢？答案是可以的。

我们首先在菜单栏中，点击窗口/首选项

选择常规/外观/颜色和字体/Java 编辑器文本字体/编辑

我们自己选择适合自己的字体以及大小，选择完成之后，我们点确定。

出来之后，我们可以在下方预览字体以及大小。如果不合适我们可以再次点击编辑更改。

如果觉得合适，我们就点击应用并关闭，保存并关闭。

下图为更改完字体后，编辑区代码的状态。

今天学习了如何下载JDK，如何配置JDK的环境变量。

下载eclipse，并将其汉化。

创建第一个程序，HelloWorld！

并且将代码编写并运行出来。

还有自定义设置编辑区的字体。
